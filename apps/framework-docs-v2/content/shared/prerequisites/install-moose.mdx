## Installing Moose CLI

<LanguageTabs>
<LanguageTabContent value="typescript">

First, ensure you have Node.js 20+ installed:

```bash
node --version  # Should be v20.0.0 or higher
```

Then install the Moose CLI globally:

```bash
npm install -g @514labs/moose-cli
```

Verify the installation:

```bash
moose --version
```

</LanguageTabContent>

<LanguageTabContent value="python">

First, ensure you have Python 3.10+ installed:

```bash
python --version  # Should be 3.10 or higher
```

Then install the Moose CLI using pip:

```bash
pip install moose-cli
```

Verify the installation:

```bash
moose --version
```

</LanguageTabContent>
</LanguageTabs>

<Callout type="info" title="Docker Required">
Moose also requires Docker Desktop to run local development services. [Download Docker](https://docker.com/get-started)
</Callout>

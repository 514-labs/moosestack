## Defining Data Models

Create a data model file in your `datamodels` directory:

```typescript
// datamodels/UserEvent.ts
export interface UserEvent {
  eventId: Key<string>;
  userId: string;
  timestamp: Date;
  eventType: string;
  properties?: Record<string, any>;
}
```

<Callout type="info">
Use the `Key<T>` type for primary keys to enable automatic indexing in ClickHouse.
</Callout>

# 514 org

Use `514 org` to inspect available organizations and switch your active organization context.

## Usage

```bash
514 org [OPTIONS] <COMMAND>
```

## Subcommands

### List

List organizations available to the current authenticated user.

```bash
514 org list
```

### Switch

Switch the active organization context.

```bash
514 org switch [ORG_REF]
```

- `[ORG_REF]`: Organization ID, slug, or name. Omit to choose interactively.

## Options

- `--api-url <API_URL>`: API base URL override (`HOSTING_CLI_API_URL`).
- `-o, --org <ORG>`: Organization name or ID.
- `--json`: Output as JSON.
- `-v, --verbose...`: Enable verbose output (`-v` debug, `-vv` trace).
- `-h, --help`: Print help.

## Examples

```bash
514 org list
514 org switch acme
514 org switch
514 --json org list
```

# 514 cli

Use `514 cli` for local CLI binary management operations.

## Usage

```bash
514 cli [OPTIONS] <COMMAND>
```

## Subcommands

### Update

Update installed CLIs (`514` and/or `moose`).

```bash
514 cli update [OPTIONS] [TARGET]
```

See the dedicated page: [`514 cli update`](/hosting/cli/update).

## Options

- `--api-url <API_URL>`: API base URL override (`HOSTING_CLI_API_URL`).
- `-o, --org <ORG>`: Organization name or ID.
- `--json`: Output as JSON.
- `-v, --verbose...`: Enable verbose output (`-v` debug, `-vv` trace).
- `-h, --help`: Print help.

## Examples

```bash
514 cli update
514 cli update 514
514 cli update moose
```

# Improving the Performance of Your Dashboards

This guide will walk you through building high-performance dashboards that can handle millions of data points without breaking a sweat.

## What You'll Learn

- How to design schemas optimized for dashboard queries
- Using materialized views to pre-aggregate data
- Best practices for time-series data in ClickHouse
- Connecting your dashboard frontend to Moose APIs

## Prerequisites

- A MooseStack project set up
- Basic understanding of data modeling
- Familiarity with your dashboard framework (React, Vue, etc.)

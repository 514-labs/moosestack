# Chat in Your App

This guide shows you how to build chat functionality in your application with powerful analytics capabilities.

## What You'll Learn

- Capturing and storing chat messages at scale
- Real-time analytics on chat data
- Building AI-powered chat features with context from your data
- Implementing chat search and retrieval

## Prerequisites

- A MooseStack project set up
- Basic understanding of streaming data
- Familiarity with your frontend framework

# Code Blocks Rendering Test

This guide tests how code blocks render across different languages and scenarios commonly used in the documentation.

## Plain Text (No Language Specification)

Code blocks without a language should render as plain text with no syntax highlighting:

```
This is plain text
No syntax highlighting should be applied
Just plain text rendering
Some more lines here
```

```sql
SELECT
  user_id,
  COUNT(*) as event_count,
  MAX(timestamp) as last_activity
FROM user_events
WHERE timestamp >= now() - INTERVAL 7 DAY
GROUP BY user_id
ORDER BY event_count DESC
LIMIT 10;
```

## Rust

```rust
use clickhouse::Client;
use serde::{Deserialize, Serialize};

#[derive(Debug, Serialize, Deserialize)]
struct UserEvent {
    user_id: String,
    timestamp: i64,
    event_type: String,
}

async fn query_events(client: &Client) -> Result<Vec<UserEvent>> {
    let events = client
        .query("SELECT * FROM user_events")
        .fetch_all::<UserEvent>()
        .await?;
    Ok(events)
}
```

# Pipelines

This guide covers how to build and manage data pipelines using MooseStack to transform, aggregate, and move data through your data warehouse.

## Overview

Data pipelines enable you to automate the flow of data from source systems through transformations and into your analytical data warehouse, ensuring data quality and consistency.

## Key Concepts

- Pipeline architecture and design
- Data transformation patterns
- Orchestration and scheduling
- Error handling and monitoring
- Performance optimization

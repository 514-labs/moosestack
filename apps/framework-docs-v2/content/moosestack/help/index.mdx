## Still Need Help?

If you can't find what you're looking for:

1. Check the [Troubleshooting Guide](/moosestack/help/troubleshooting) for common issues
2. Search existing [GitHub issues](https://github.com/514-labs/moose/issues) for similar problems
3. Join our [Slack community](https://join.slack.com/t/moose-community/shared_invite/zt-2fjh5n3wz-cnOmM9Xe9DYAgQrNu8xKxg) and ask for help
4. Open a new GitHub issue with:
   - Moose version (`moose --version`)
   - Error messages and logs
   - Steps to reproduce
   - Expected vs. actual behavior

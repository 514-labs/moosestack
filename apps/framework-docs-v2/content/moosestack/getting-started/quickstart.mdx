# 5-Minute Quickstart

This quickstart creates a new project with the canonical multi-engine contract.

## 1. Install CLI

```bash
bash -i <(curl -fsSL https://fiveonefour.com/install.sh) moose
```

## 2. Create a project

```bash
moose init my-analytics-app typescript --olap-engine clickhouse
cd my-analytics-app
pnpm install
```

## 3. Configure engine and profile

```toml title="moose.config.toml"
[olap]
engine = "clickhouse" # clickhouse | duckdb | snowflake
profile = "local"
connection = "http://panda:pandapass@localhost:18123/local"

[dev_infra]
profile = "local" # local | remote_hybrid | remote_full
```

## 4. Start development

```bash
moose dev --infra-profile local
```

## 5. Create and apply a migration plan

```bash
export MOOSE_OLAP__CONNECTION="http://panda:pandapass@localhost:18123/local"

moose generate migration --olap-engine clickhouse --olap-url "$MOOSE_OLAP__CONNECTION" --save
moose migrate --olap-engine clickhouse --olap-url "$MOOSE_OLAP__CONNECTION"
```

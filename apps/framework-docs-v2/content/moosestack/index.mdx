# MooseStack

MooseStack lets you build analytical backends with one code-first contract across ClickHouse, DuckDB, and Snowflake.

## Start here

- [5-Minute Quickstart](/moosestack/getting-started/quickstart)
- [Remote-First Quickstart](/moosestack/getting-started/remote-first-quickstart)
- [Start from an Existing Warehouse](/moosestack/getting-started/from-existing-warehouse)

## Core contract

- Configure engine and profile in [`[olap]` and `[dev_infra]`](/moosestack/configuration/engine-and-dev-profile).
- Run development with `moose dev --infra-profile ...`.
- Plan and apply migrations with engine-aware flags.

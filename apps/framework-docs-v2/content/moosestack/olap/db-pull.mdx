# DB Pull

`moose db pull` refreshes models marked as `EXTERNALLY_MANAGED`.

## Command

```bash
moose db pull --olap-engine <clickhouse|duckdb|snowflake> --olap-url <URL> [--file-path <OUTPUT_PATH>]
```

## Examples

```bash
moose db pull --olap-engine clickhouse --olap-url "https://user:password@host:8443/db"
moose db pull --olap-engine duckdb --olap-url "duckdb:///./warehouse.duckdb"
moose db pull --olap-engine snowflake --olap-url "snowflake://user:password@account/db/schema?warehouse=wh"
```

## What it updates

- External model definitions in your generated external models file.
- New externally managed resources discovered from target warehouse.

## What it never does

- It does not mutate remote schema.
- It does not update fully managed models.

## Common errors

- `connection_error`: URL or credentials are invalid.
- `capability_mismatch`: feature is not supported by selected engine.

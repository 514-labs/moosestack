# Migrations

Moose migrations keep code and warehouse schema in sync.

## Core workflow

1. Run `moose generate migration --olap-engine ... --olap-url ... --save`.
2. Review `migrations/plan.yaml`.
3. Run `moose migrate --olap-engine ... --olap-url ...`.

# Dev Environment Configuration

Use this page to tune the developer workflow for local or remote infrastructure.

## Core profile setting

```toml
[dev_infra]
profile = "local" # local | remote_hybrid | remote_full
```

## Lifecycle hooks

```toml
[http_server_config]
on_first_start_script = "moose db pull --olap-engine clickhouse --olap-url $MOOSE_OLAP__CONNECTION"
on_reload_complete_script = "echo 'reload complete'"
```

## Useful dev commands

```bash
moose dev --infra-profile local
moose dev --infra-profile remote_hybrid
moose dev --infra-profile remote_full
```

## Local service ports

| Port    | Service               | Config key                           |
| ------- | --------------------- | ------------------------------------ |
| `4000`  | API server            | `http_server_config.port`            |
| `5001`  | Management API        | `http_server_config.management_port` |
| `19092` | Redpanda local broker | `redpanda_config.broker`             |
| `8080`  | Temporal UI           | `temporal_config.ui_port`            |

## Diagnostics

- `connection_error`: external service is unreachable.
- `profile_misconfiguration`: profile and connection settings conflict.

# ClickHouse Engine Profile

This page shows the ClickHouse-specific profile using the canonical engine-neutral contract.

## Recommended config

```toml
[olap]
engine = "clickhouse"
profile = "remote"
connection = "https://user:password@host:8443/db"

[dev_infra]
profile = "remote_hybrid"
```

## Environment variables

```bash
MOOSE_OLAP__ENGINE=clickhouse
MOOSE_OLAP__PROFILE=remote
MOOSE_OLAP__CONNECTION=https://user:password@host:8443/db
MOOSE_DEV_INFRA__PROFILE=remote_hybrid
```

## Typical command set

```bash
moose dev --infra-profile remote_hybrid
moose plan --olap-engine clickhouse --olap-url "$MOOSE_OLAP__CONNECTION"
moose migrate --olap-engine clickhouse --olap-url "$MOOSE_OLAP__CONNECTION"
```

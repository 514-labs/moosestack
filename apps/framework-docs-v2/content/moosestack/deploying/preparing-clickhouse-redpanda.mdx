# Prepare ClickHouse Cloud and Redpanda Cloud

Use this checklist before you run Moose in remote-first mode.

## ClickHouse Cloud

Set engine-neutral OLAP variables:

```bash
MOOSE_OLAP__ENGINE=clickhouse
MOOSE_OLAP__PROFILE=remote
MOOSE_OLAP__CONNECTION=https://user:password@host:8443/db
```

## Redpanda Cloud

```bash
MOOSE_REDPANDA_CONFIG__BROKER=<broker-host>:9092
MOOSE_REDPANDA_CONFIG__SECURITY_PROTOCOL=SASL_SSL
MOOSE_REDPANDA_CONFIG__SASL_MECHANISM=SCRAM-SHA-256
MOOSE_REDPANDA_CONFIG__SASL_USERNAME=<username>
MOOSE_REDPANDA_CONFIG__SASL_PASSWORD=<password>
MOOSE_REDPANDA_CONFIG__NAMESPACE=<team-or-branch>
```

## Validate before deployment

```bash
moose plan --olap-engine clickhouse --olap-url "$MOOSE_OLAP__CONNECTION"
```

If validation fails, check diagnostics output first.

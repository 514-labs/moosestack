# Moose CLI Reference

## Install

```bash
bash -i <(curl -fsSL https://fiveonefour.com/install.sh) moose
```

## Core commands

### Init

```bash
moose init <name> <typescript|python> [--olap-engine <clickhouse|duckdb|snowflake>]
```

### Dev

```bash
moose dev [--infra-profile <local|remote_hybrid|remote_full>] [--mcp] [--timestamps] [--timing] [--log-payloads]
```

### Plan

```bash
moose plan --olap-engine <clickhouse|duckdb|snowflake> --olap-url <URL>
```

### Generate migration

```bash
moose generate migration --olap-engine <clickhouse|duckdb|snowflake> --olap-url <URL> --save
```

### Apply migration

```bash
moose migrate --olap-engine <clickhouse|duckdb|snowflake> --olap-url <URL>
```

### DB pull

```bash
moose db pull --olap-engine <clickhouse|duckdb|snowflake> --olap-url <URL> [--file-path <OUTPUT_PATH>]
```

### Common utilities

```bash
moose check
moose build
moose clean
moose logs --tail
moose ls
moose query "SELECT 1"
```

## Diagnostics categories

- `connection_error`
- `profile_misconfiguration`
- `capability_mismatch`
- `migration_contract_error`
